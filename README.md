# Furute Website

## Overview
A modern Next.js website for Furute featuring programs, events,
consultancy services, branding, digital marketing, mentoring,
speaker invitations, and contact forms.

## Tech Stack

- Next.js 16
- TypeScript
- React
- MongoDB
- Mongoose
- Jest
- GitHub Actions
- Vercel Ready

## Features

- Responsive Design
- Programs Module
- Events Module
- Services Module
- Contact API
- Speaker API
- MongoDB Integration
- Jest Testing

## Installation

git clone ...

npm install

npm run dev

## Testing

npm test

## Production Build
