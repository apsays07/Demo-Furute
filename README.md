# 🚀 Furute Website

A modern, responsive business consulting and leadership development platform built with **Next.js 16**, **React 19**, **TypeScript**, and **MongoDB**.

---

## 📖 Overview

Furute is a professional corporate website designed to showcase consulting services, leadership programs, training events, mentoring initiatives, branding solutions, digital marketing services, and speaker invitations.

The application is optimized for performance, accessibility, SEO, and deployment on Vercel.

---

## ✨ Features

* Responsive UI
* Leadership Programs
* Events Module
* Consultancy Services
* Branding & Digital Marketing
* Mentoring Programs
* Speaker Invitation Form
* Contact Form
* MongoDB Integration
* REST API Routes
* Jest Unit & Integration Testing
* GitHub Actions CI
* Vercel Ready

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
cd furute-next-js
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

---

## 🧪 Testing

Run Jest tests:

```bash
npm test
```

Generate production build:

```bash
npm run build
```

---

## 🔐 Environment Variables

Create a `.env.local` file:

```
MONGODB_URI=your_mongodb_connection_string
```

Do **not** commit `.env.local`.

---

## 🚀 Deployment

This project is optimized for deployment on **Vercel**.

Deployment checklist:

* npm test
* npm run build
* Configure environment variables
* Connect GitHub repository
* Deploy
